# Documentation for Dispatch Visual Novel: Episode 1 (Robert's Pivot)
This document explains how the C++ program for Dispatch Visual Novel: Episode 1 (Robert's Pivot) works. The program is a simple visual novel that simulates Robert's journey from superhero to dispatcher through 5 decision points, using C++ concepts like functions, Booleans, integers, and conditional statements. It incorporates dialogue and choices to create branching narratives, with variable tracking character states for the epilogue.
